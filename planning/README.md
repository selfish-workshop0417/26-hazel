# 기획 (Planning)

이 폴더는 레퍼런스랩의 "무엇을 만들 것인가"의 상세를 담습니다.

---

## 1. 핵심 기능 명세

### 기능 1. 인스타그램 콘텐츠 후킹포인트 분석

**동작 흐름:**
1. 사용자가 인스타그램 게시물/릴스 링크를 입력
2. 이미지/영상 전체를 크롤링 및 분석
3. 아래 항목에 대한 분석 결과 제공

**분석 항목:**
- **후킹포인트**: 시청자의 시선을 끄는 첫 장면/문구/구성 요소
- **카피 분석**: 텍스트 문구, 톤앤매너, 감정 유발 키워드
- **영상 구성**: 첫 화면부터 마지막 화면까지 장면 흐름 및 구성 분석 (텍스트 + 비주얼 요소 포함)
- **USP (Unique Selling Point)**: 해당 콘텐츠가 경쟁 콘텐츠와 차별화되는 핵심 포인트

**결과 표시 방식:**
- 항목별 텍스트 요약 형식으로 제공
- 히스토리 보관: 분석했던 링크와 결과를 나중에 다시 볼 수 있도록 저장
- 복사/내보내기: 분석 결과를 텍스트로 복사하거나 파일로 내보내기 가능
- 폴더/태그 분류: 캠페인별, 브랜드별로 분류해서 정리 가능

---

### 기능 2. 핀터레스트 → 피그마 자동 수집

**동작 흐름:**
1. 사용자가 특정 핀터레스트 보드를 지정 (최초 1회 연동)
2. 해당 보드에 이미지가 저장되면 자동으로 피그마에 싱크
3. 핀터레스트 보드 하나당 피그마 페이지 하나로 분리되어 이미지 자동 배치

**연동 방식:**
- 핀터레스트 특정 보드 URL 기반 자동 싱크
- 신규 저장 이미지 감지 → 피그마 자동 업데이트
- 보드별로 피그마 페이지를 분리하여 정리 (예: 보드A → 피그마 Page 1, 보드B → 피그마 Page 2)

## 2. 사용자 시나리오

### 시나리오 A — 광고 소재 기획 전 레퍼런스 분석

- **Who**: 광고 소재를 기획하는 콘텐츠 마케터
- **When**: 새 캠페인 시작 전 레퍼런스 수집 단계
- **Where**: 업무 환경 (데스크톱)
- **Why**: 경쟁사 및 트렌드 광고의 후킹포인트를 빠르게 파악하기 위해
- **How**: 인스타그램에서 참고할 광고 링크 복사 → 레퍼런스랩에 붙여넣기 → 후킹포인트/카피/USP 분석 결과 확인 → 폴더에 저장 → 소재 기획에 반영

### 시나리오 B — 비주얼 레퍼런스 피그마 정리

- **Who**: 광고 소재를 제작하는 PD
- **When**: 시안 작업 전 무드보드 구성 단계
- **Where**: 업무 환경 (데스크톱)
- **Why**: 핀터레스트에서 수집한 이미지를 피그마로 옮기는 반복 작업을 줄이기 위해
- **How**: 핀터레스트 보드에 이미지 저장 → 자동으로 피그마 해당 페이지에 싱크 → 바로 무드보드 작업 시작

## 3. MVP 범위

### ✅ MVP에 포함 (필수 — 두 기능 동시 개발)
- 인스타그램 링크 입력 → 후킹포인트 / 카피 / 영상 구성 / USP 분석 (항목별 텍스트 요약)
- 분석 결과 히스토리 보관 / 복사·내보내기 / 폴더·태그 분류
- 핀터레스트 특정 보드 → 피그마 자동 싱크 (보드별 페이지 분리)

### 🔜 추후 고려 (v2)
- 여러 링크 일괄 분석 (배치 처리)
- 분석 결과 팀 공유 기능
- 유튜브 / 틱톡 등 타 플랫폼 확장

---

## 4. 우선순위

| 순위 | 기능 | 이유 |
|------|------|------|
| 1순위 | 인스타그램 후킹포인트 분석 + 결과 저장/분류 | 핵심 가치, 즉시 업무 활용 가능 |
| 1순위 | 핀터레스트 → 피그마 자동 싱크 | 반복 작업 제거, 효율 극대화 |
| 2순위 | 배치 분석 / 팀 공유 | MVP 이후 추가 |
