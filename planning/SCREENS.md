# 화면 기획 (Screen Planning)

디자인팀 전달용 화면 상세 기획서입니다.

---

## 전체 화면 목록

| 화면 | 주요 기능 |
|------|---------|
| 홈 | 레퍼런스 수집 / 분석 선택 |
| 분석 | 링크 입력 → 결과 스크롤 → 태그 저장 |
| 수집 | 공개 보드 URL → 피그마 연동 → 싱크 |
| 보관함 | 태그 필터 + 리스트형 분석 목록 |
| 설정 | 피그마 API 키 등 연동 관리 |

---

## 사용자 플로우

```
홈 (수집 OR 분석 선택)
  ├── 레퍼런스 분석 선택
  │     └── 인스타그램 링크 입력
  │           └── 분석 결과 확인 (스크롤)
  │                 └── 저장 버튼 → 태그 입력 → 보관함 저장
  └── 레퍼런스 수집 선택
        └── 핀터레스트 공개 보드 URL 입력
              └── 피그마 연동 설정
                    └── 싱크 시작 → 자동 수집
```

---

## 화면별 상세

### 1. 홈
- 레퍼런스랩 로고/타이틀
- **레퍼런스 분석** 버튼
- **레퍼런스 수집** 버튼

---

### 2. 분석 화면

**입력 영역**
- 인스타그램 링크 입력창
- 분석 시작 버튼
- 분석 중 로딩 상태

**결과 영역 (스크롤)**
1. 후킹포인트
2. 카피 분석
3. 영상 구성 (장면별 흐름)
4. USP (Unique Selling Point)

**저장 영역**
- 저장 버튼
- 태그 입력창 (기존 태그 자동완성 지원)
- 저장 완료 → 보관함으로 이동

---

### 3. 수집 화면 (핀터레스트→피그마)

**연동 설정**
- 핀터레스트 공개 보드 URL 입력창
- 피그마 API 키 입력창
- 싱크 시작 버튼

**연동 현황**
- 연동된 보드 목록 (보드명 + 연결된 피그마 페이지명)
- 마지막 싱크 시간

> **MVP 범위**: 공개 보드만 지원. 비공개 보드(핀터레스트 로그인 연동)는 v2에서 추가.

---

### 4. 보관함

**필터 영역**
- 태그 목록 (상단 가로 스크롤)
- 태그 클릭 시 필터링

**리스트 영역**
- 인스타그램 썸네일 + 분석 요약 텍스트
- 클릭 시 분석 결과 상세 페이지로 이동

---

### 5. 설정

- 피그마 API 키 관리
- 연동된 핀터레스트 보드 관리
- (v2) 핀터레스트 계정 로그인 연동

---

## 디자인팀 전달 시 참고사항

- 회원가입/로그인 없음 (MVP)
- 회사명/제품명 입력 없음 (MVP)
- 모든 데이터는 로컬 또는 단일 사용자 기준으로 관리
- 태그는 사용자가 자유롭게 추가 가능, 기존 태그 자동완성 지원
