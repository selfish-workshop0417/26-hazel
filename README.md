# 레퍼런스랩 (ReferenceLab)

인스타그램/핀터레스트 콘텐츠를 분석하고 레퍼런스를 수집·정리해주는 서비스

## 프로젝트 정보
- **유형**: SaaS (도구)
- **타겟 사용자**: 콘텐츠 마케터 및 PD (광고 소재 제작)
- **출시 목표**: 업무용 내부 도구

## 핵심 기능
- 인스타그램 링크 입력 → 이미지/영상 후킹포인트 자동 분석
- 핀터레스트 보드 이미지 → 피그마 자동 수집

## 폴더 구조 (5개 팀)
- `planning/` — 기획 팀: 무엇을 만들 것인가
- `design/` — 디자인 팀: 어떻게 보여줄 것인가
- `cx/` — CX 팀: 누가 어떻게 사용할 것인가
- `marketing/` — 마케팅 팀: 어떻게 알릴 것인가
- `strategy/` — 전략 팀: 어떻게 지속 가능하게 만들 것인가
